# GitHub-Scraper

This project will get the top 5 issues from the top 5 projects on the top 3 topics on GitHub and save it in a JSON file; promoting hassle-free issue search and easier contributions to open-source.

Prerequisites:
- cheerio package <br>
``` $ npm i cheerio ```
- request package <br>
``` $ npm i request ```
- fs package <br>
``` $ npm i fs ```

Happy coding!
